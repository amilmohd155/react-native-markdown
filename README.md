# @quizcript/react-native-markdown

Markdown renderer for React Native

## Installation

```sh
npm install @quizcript/react-native-markdown
```

## Usage


```js
import { multiply } from '@quizcript/react-native-markdown';

// ...

const result = await multiply(3, 7);
```
